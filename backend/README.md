# Backend

## Ubicacion real del backend
El backend funcional ya fue movido a `backend/api`.

## Que hay en `backend/api`
- Laravel completo
- API REST
- autenticacion
- productos
- pedidos
- pagos
- panel admin
- configuracion de base de datos

## Como ejecutarlo
1. Entra a `backend/api`
2. Ejecuta `php artisan serve`
3. La API quedara disponible por defecto en `http://127.0.0.1:8000`

## Archivos importantes
- `backend/api/routes/api.php`
- `backend/api/app/Http/Controllers`
- `backend/api/app/Models`
- `backend/api/database/migrations`
- `backend/api/.env`
