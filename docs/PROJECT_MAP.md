## Estructura funcional del sistema
- Cliente web: catalogo, carrito, pedidos, seguimiento.
- Admin web: productos, pedidos, usuarios.
- API Laravel: autenticacion, productos, pedidos, validacion.
- PostgreSQL: persistencia principal.
